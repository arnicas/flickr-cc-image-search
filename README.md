
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
   `npm run dev`

Be sure to set the `FLICKR_API_KEY` in [.env.local](.env.local) to your Flickr API key.

## Deploy

1. Build the app:
   `npm run build`
